# SajiloForm (सजिलोForm) 🚀

An embeddable AI-powered autofill SDK designed to read Nepalese identity documents (Citizenship, National ID) natively in Devanagari script and automatically populate online forums, banking registration setups, and government portals.

## Repository Structure
- `backend/`: FastAPI app processing document extractions (Port 8001).
- `demo-portal/`: React + Vite mock platform showcasing the SDK autofill functionality (Port 5173).
